# LinkedIn Profile Builder

Build a complete, professional LinkedIn profile through conversational interview combined with automated credential mining.

## Process

Phase 1: Discovery Interview — broad open-ended questions adapting based on answers
Phase 2: Credential Mining — parallel web searches for press, achievements, collaborations
Phase 3: Profile Drafting — headline, about, experience, skills, featured section
Phase 4: Review and Delivery — iterate until it sounds like the user

## Anti-Patterns

- No corporate zombie voice
- No underselling — dig deeper on every achievement
- No padding — shorter and punchier beats inflated
- Match the user's industry voice, not a template
