# Contributing

This plugin was built through Adversarial Multi-AI Consensus Building (AMACB) then stress-tested by six simulated Anthropic engineering teams. Rather than picking a winner, we integrated all six perspectives.

## The Six Perspectives

| Team | Contribution | Location |
|------|-------------|----------|
| Trust and Safety (Shield) | Bilateral consent, verification seals | skills/verification-seal/ |
| Legal (Factual) | FCRA disclaimers, citation export | references/legal-framework.md |
| ML Engineering (Calibrated) | Bayesian confidence model | references/calibration-model.md |
| Product (Shine) | 5-question quick build | skills/profile-builder/references/quick-build-mode.md |
| Security (Lockbox) | Audit logging, rate limits | references/security-framework.md |
| Contrarian (Raw) | This document. Ship and iterate in public. |

## How to Contribute

Fork, build your version, submit a PR. If you disagree with a design decision, write the alternative. The best ideas emerge from genuine adversarial pressure, not polite suggestions. This is how the plugin was built. This is how it should evolve.
