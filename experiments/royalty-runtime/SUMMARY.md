Royalty Runtime is an execution observability system that evolved from enforcement-first design to observability-first strategy.

Key principles:
- attribution before payment
- store first, verify clearly, interpret later
- pay packages before people
- enforcement gets forked, measurement gets embedded

System pieces:
- lineage extraction
- deterministic hashing
- collector ledger
- attribution weighting
- optional future royalty routing
