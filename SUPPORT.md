## Getting Help

**For questions, use [GitHub Discussions](https://github.com/splitmerge420/uws/discussions)** — not GitHub Issues.

| Topic | Where to go |
|---|---|
| Question about how to use a command | [Discussions → Q&A](https://github.com/splitmerge420/uws/discussions/categories/q-a) |
| Feature idea or proposal | [Discussions → Ideas](https://github.com/splitmerge420/uws/discussions/categories/ideas) |
| Bug report | [Open an Issue](https://github.com/splitmerge420/uws/issues/new/choose) |
| Security vulnerability | [Private Security Advisory](https://github.com/splitmerge420/uws/security/advisories/new) |

## Response Time

This is a community-maintained open-source project. Response times vary:

- **Security vulnerabilities**: Within 48 hours
- **Bug reports**: Typically within a week
- **Feature requests**: No SLA — prioritised by community interest

## Commercial Support

Commercial support, SLA-backed deployments, and custom integrations are available through **[uws.run](https://github.com/splitmerge420/uws)**. Contact the maintainer via GitHub.
